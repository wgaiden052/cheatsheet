Common Table Tags
Tag Name Description
<table> Table Head The wrapper element for all HTML tables
<th> Table Head The set of rows defining the column header in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row Defines a row in a table
<td> Table Data The contents of each cell within a row
<tfoot> Table Footer The set of rows defining the footer for a table

Various HTML Syntax Notes

  1. Use the '&' to assign tags to characters you can't regularly type (example: &+lt = <)
  2. To import an image, use '<img src="___">'
  3. To link your css page, use '<link rel="stylesheet" type"text/css" href="--link--">'
